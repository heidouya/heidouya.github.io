# HTTP жҳҜд»Җд№Ҳ

HTTP жҳҜдёҖз§ҚйҖҡдҝЎеҚҸи®®пјҢе®ғе…Ғи®ёе°Ҷи¶…ж–Үжң¬ж Үи®°иҜӯиЁҖпјҲHTMLпјүж–ҮжЎЈд»Һ web жңҚеҠЎеҷЁдј йҖҒеҲ°е®ўжҲ·з«Ҝзҡ„жөҸи§ҲеҷЁгҖӮ

HTTP жҳҜдёҖдёӘеұһдәҺеә”з”ЁеұӮзҡ„йқўеҗ‘еҜ№иұЎзҡ„еҚҸи®®пјҢз”ұдәҺе…¶з®ҖжҚ·гҖҒеҝ«йҖҹзҡ„ж–№ејҸпјҢйҖӮз”ЁдәҺеҲҶеёғејҸи¶…еӘ’дҪ“дҝЎжҒҜзі»з»ҹгҖӮе®ғдёҺ 1990 е№ҙжҸҗеҮәпјҢз»ҸиҝҮеҮ е№ҙзҡ„дҪҝз”ЁдёҺеҸ‘еұ•пјҢеҫ—еҲ°дёҚж–ӯе®Ңе–„е’Ңжү©еұ•гҖӮ

# Web е’Ң HTTP зҡ„е…ізі»

Web жҳҜдёҖз§ҚеҹәдәҺи¶…ж–Үжң¬е’Ң HTTP зҡ„гҖҒе…ЁзҗғжҖ§зҡ„гҖҒеҠЁжҖҒдәӨдә’зҡ„гҖҒи·Ёе№іеҸ°зҡ„еҲҶеёғејҸеӣҫеҪўдҝЎжҒҜзі»з»ҹгҖӮ

зҪ‘з«ҷжҳҜе»әз«ӢеңЁ Internet дёҠзҡ„дёҖз§ҚзҪ‘з»ңжңҚеҠЎпјҢдёәжөҸи§ҲиҖ…еңЁ Internet дёҠжҹҘжүҫе’ҢжөҸи§ҲдҝЎжҒҜжҸҗдҫӣдәҶеӣҫеҪўеҢ–зҡ„гҖҒжҳ“дәҺи®ҝй—®зҡ„зӣҙи§ӮйЎөйқўпјҢе…¶дёӯзҡ„ж–ҮжЎЈеҸҠи¶…зә§й“ҫжҺҘе°Ҷ Internet дёҠзҡ„дҝЎжҒҜиҠӮзӮ№з»„з»ҮжҲҗдёҖдёӘдә’дёәе…іиҒ”зҡ„зҪ‘зҠ¶з»“жһ„гҖӮ

# HTTPеҚҸи®®зҡ„еүҚдё–д»Ҡз”ҹ

HTTP0.9пјҢеҸӘжңүдёҖдёӘ GET иҜ·жұӮж–№жі•пјҢжІЎжңү HEAD зӯүжҸҸиҝ°гҖӮ

иҜ·жұӮзҡ„ж—¶еҖҷпјҡ

GET + иҜ·жұӮзҡ„ж–Үд»¶и·Ҝеҫ„

жңҚеҠЎз«Ҝж”¶еҲ°иҜ·жұӮеҗҺиҝ”еӣһдёҖдёӘд»Ҙ ASCII еӯ—з¬ҰжөҒзј–з Ғзҡ„ HTML ж–ҮжЎЈпјҢжңҚеҠЎеҷЁеҸ‘йҖҒе®ҢжҜ•еҗҺе°ұдјҡе…ій—ӯ TCP иҝһжҺҘгҖӮ

HTTP1.0

  • еўһеҠ дәҶ HEADгҖҒPOST зӯүж–°ж–№жі•
  • еўһеҠ дәҶе“Қеә”зҠ¶жҖҒз ҒпјҢж Үи®°еҸҜиғҪзҡ„й”ҷиҜҜеҺҹеӣ 
  • еј•е…ҘдәҶеҚҸи®®зүҲжң¬еҸ·жҰӮеҝө
  • еј•е…ҘдәҶ HTTP HeaderпјҲеӨҙйғЁпјүзҡ„жҰӮеҝөпјҢи®©HTTPеӨ„зҗҶиҜ·жұӮе’Ңе“Қеә”жӣҙеҠ зҒөжҙ»
  • дј иҫ“зҡ„ж•°жҚ®дёҚеҶҚеұҖйҷҗдәҺж–Үжң¬

HTTP1.1

  • й•ҝиҝһжҺҘпјҡеј•е…ҘдәҶ TCP иҝһжҺҘеӨҚз”ЁпјҢеҚідёҖдёӘ TCP й»ҳи®ӨдёҚе…ій—ӯпјҢеҸҜд»Ҙиў«еӨҡдёӘиҜ·жұӮеӨҚз”Ё
  • е№¶еҸ‘иҝһжҺҘпјҡеҜ№дёҖдёӘеҹҹеҗҚзҡ„иҜ·жұӮе…Ғи®ёеҲҶй…ҚеӨҡдёӘй•ҝиҝһжҺҘпјҲзј“и§ЈдәҶй•ҝиҝһжҺҘдёӯзҡ„гҖҢйҳҹеӨҙйҳ»еЎһгҖҚй—®йўҳпјү еј•е…Ҙз®ЎйҒ“жңәеҲ¶пјҢдёҖдёӘ TCP иҝһжҺҘпјҢеҸҜд»ҘеҗҢж—¶еҸ‘йҖҒеӨҡдёӘиҜ·жұӮгҖӮпјҲе“Қеә”зҡ„йЎәеәҸеҝ…йЎ»е’ҢиҜ·жұӮзҡ„йЎәеәҸдёҖиҮҙпјҢеӣ жӯӨдёҚеёёз”Ёпјү
  • еўһеҠ дәҶ PUTгҖҒDELETEгҖҒOPTIONSгҖҒPATCH зӯүж–°зҡ„ж–№жі•
  • ж–°еўһдәҶдёҖдәӣзј“еӯҳзҡ„еӯ—ж®өпјҲIf-Modified-Since, If-None-Matchпјү
  • иҜ·жұӮеӨҙдёӯеј•е…ҘдәҶ range еӯ—ж®өпјҢж”ҜжҢҒж–ӯзӮ№з»ӯдј 
  • е…Ғи®ёе“Қеә”ж•°жҚ®еҲҶеқ—пјҲchunkedпјүпјҢеҲ©дәҺдј иҫ“еӨ§ж–Үд»¶ ејәеҲ¶иҰҒжұӮ Host еӨҙпјҢи®©дә’иҒ”зҪ‘дё»жңәжүҳз®Ўз§°дёәеҸҜиғҪ

HTTP2

  • ж•°жҚ®ж”№дёәдәҢиҝӣеҲ¶дј иҫ“гҖӮ
  • еӨҡи·ҜеӨҚз”ЁпјҢеҗҢдёҖдёӘиҝһжҺҘйҮҢеҸҜд»ҘеҸ‘йҖҒеӨҡдёӘиҜ·жұӮпјҢе№¶дё”еҸҜд»ҘдёҚжҢүйЎәеәҸжқҘгҖӮ
  • дҪҝз”Ёдё“з”Ёз®—жі•еҺӢзј©еӨҙйғЁпјҢеҮҸе°‘ж•°жҚ®дј иҫ“йҮҸ
  • е…Ғи®ёжңҚеҠЎеҷЁдё»еҠЁеҗ‘е®ўжҲ·жҺЁйҖҒж•°жҚ®

HTTP3(QUIC еҚҸи®®)

HTTP/3 е°Ҷеә•еұӮдҫқиө–зҡ„ TCP ж”№жҲҗ UDPпјҢUDP зӣёеҜ№дәҺ TCP иҖҢиЁҖжңҖеӨ§зҡ„зү№зӮ№жҳҜдј иҫ“ж•°жҚ®ж—¶дёҚйңҖиҰҒе»әз«ӢиҝһжҺҘпјҢеҸҜд»ҘеҗҢж—¶еҸ‘йҖҒеӨҡдёӘж•°жҚ®еҢ…пјҢжүҖд»Ҙдј иҫ“ж•ҲзҺҮеҫҲй«ҳпјҢзјәзӮ№е°ұжҳҜжІЎжңүзЎ®и®ӨжңәеҲ¶жқҘдҝқиҜҒеҜ№ж–№дёҖе®ҡиғҪж”¶еҲ°ж•°жҚ®гҖӮ

# йҖҸиҝҮTCPIPзңӢHTTP

HTTP еҚҸи®®жҳҜжһ„е»әеңЁ TCP/IP еҚҸи®®д№ӢдёҠзҡ„пјҢжҳҜ TCP/IP еҚҸи®®зҡ„дёҖдёӘеӯҗйӣҶгҖӮдёәдәҶжӣҙеҘҪзҡ„зҗҶи§Ј HTTP еҚҸи®®пјҢжҲ‘д»¬е…ҲдәҶи§ЈдёҖдёӢ TCP/IP зҡ„зӣёе…ізҹҘиҜҶгҖӮ

TCP/IP еҚҸи®®е…¶е®һжҳҜдёҖзі»еҲ—дёҺдә’иҒ”зҪ‘зӣёе…іиҒ”зҡ„еҚҸи®®йӣҶеҗҲзҡ„жҖ»з§°гҖӮеҲҶеұӮз®ЎзҗҶжҳҜ TCP/IP еҚҸи®®зҡ„йҮҚиҰҒзү№еҫҒгҖӮ

TCP/IP еҚҸи®®ж—ҸжҳҜз”ұдёҖдёӘеӣӣеұӮеҚҸи®®з»„жҲҗзҡ„зі»з»ҹпјҢиҝҷеӣӣеұӮеҲҶеҲ«жҳҜпјҡеә”з”ЁеұӮгҖҒдј иҫ“еұӮгҖҒзҪ‘з»ңеұӮгҖҒж•°жҚ®й“ҫи·ҜеұӮгҖӮ

еә”з”ЁеұӮпјҡ

еә”з”ЁеұӮдёҖиҲ¬жҳҜжҲ‘д»¬зј–еҶҷзҡ„еә”з”ЁзЁӢеәҸпјҢеҶіе®ҡдәҶеҗ‘з”ЁжҲ·жҸҗдҫӣзҡ„еә”з”ЁжңҚеҠЎгҖӮеә”з”ЁеұӮеҸҜд»ҘйҖҡиҝҮзі»з»ҹи°ғз”ЁдёҺдј иҫ“еұӮиҝӣиЎҢйҖҡдҝЎгҖӮеҰӮпјҡFTPгҖҒDNSгҖҒHTTP зӯүгҖӮ

дј иҫ“еұӮпјҡ

дј иҫ“еұӮйҖҡиҝҮзі»з»ҹи°ғз”Ёеҗ‘еә”з”ЁеұӮжҸҗдҫӣеӨ„дәҺзҪ‘з»ңиҝһжҺҘдёӯзҡ„дёӨеҸ°и®Ўз®—жңәд№Ӣй—ҙзҡ„ж•°жҚ®дј иҫ“еҠҹиғҪгҖӮ

еңЁдј иҫ“еұӮдёӯжңүдёӨдёӘжҖ§иҙЁдёҚеҗҢзҡ„еҚҸи®®пјҡTCP е’Ң UDPгҖӮTCP жҳҜйқўеҗ‘иҝһжҺҘзҡ„пјҢUDP жҳҜж— иҝһжҺҘзҡ„гҖӮеӣ дёә TCP йңҖиҰҒе»әз«ӢиҝһжҺҘжүҖд»ҘжҜ”иҫғеҸҜйқ пјҢдҪҶжҳҜж•ҲзҺҮзӣёеҜ№иҫғдҪҺгҖӮUDP дёҚйңҖиҰҒе»әз«ӢиҝһжҺҘпјҢдј иҫ“ж•ҲзҺҮжҜ”иҫғй«ҳпјҢдҪҶеӣ дёәж— иҝһжҺҘпјҢжүҖд»Ҙе®үе…ЁжҖ§зӣёеҜ№иҫғдҪҺгҖӮ

зҪ‘з»ңеұӮпјҡ

зҪ‘з»ңеұӮз”ЁжқҘеӨ„зҗҶеңЁзҪ‘з»ңдёҠжөҒеҠЁзҡ„ж•°жҚ®еҢ…пјҢж•°жҚ®еҢ…жҳҜзҪ‘з»ңдј иҫ“зҡ„жңҖе°Ҹж•°жҚ®еҚ•дҪҚгҖӮиҜҘеұӮи§„е®ҡдәҶйҖҡиҝҮжҖҺж ·зҡ„и·Ҝеҫ„пјҲдј иҫ“и·ҜзәҝпјүеҲ°иҫҫеҜ№ж–№и®Ўз®—жңәпјҢе№¶жҠҠж•°жҚ®еҢ…дј иҫ“з»ҷеҜ№ж–№гҖӮ

й“ҫи·ҜеұӮпјҡ

й“ҫи·ҜеұӮз”ЁжқҘеӨ„зҗҶиҝһжҺҘзҪ‘з»ңзҡ„зЎ¬д»¶йғЁеҲҶпјҢеҢ…жӢ¬жҺ§еҲ¶ж“ҚдҪңзі»з»ҹгҖҒзЎ¬д»¶и®ҫеӨҮй©ұеҠЁгҖҒNICпјҲNetwork Interface Card зҪ‘з»ңйҖӮй…ҚеҷЁпјүд»ҘеҸҠе…үзәӨзӯүзү©зҗҶеҸҜи§ҒйғЁеҲҶгҖӮзЎ¬д»¶дёҠзҡ„иҢғз•ҙеқҮеңЁй“ҫи·ҜеұӮзҡ„дҪңз”ЁиҢғеӣҙеҶ…гҖӮ

# ж•°жҚ®еҢ…зҡ„е°ҒиЈ…иҝҮзЁӢ

дёҠеұӮеҚҸи®®зҡ„ж•°жҚ®жҳҜжҖҺд№ҲиҪ¬еҸҳжҲҗдёӢеұӮеҚҸи®®ж•°жҚ®зҡ„пјҹ

еә”з”ЁзЁӢеәҸеңЁж•°жҚ®еңЁеҸ‘еёғеҲ°ж•°жҚ®зҪ‘з»ңеүҚдјҡжІҝзқҖеҚҸи®®ж Ҳд»ҺдёҠеҫҖдёӢеӨ„зҗҶпјҢжҜҸеұӮеҚҸи®®йғҪе°ҶеңЁдёҠеұӮеҚҸи®®зҡ„еҹәзЎҖдёҠеҠ дёҠиҮӘе·ұзҡ„еӨҙйғЁдҝЎжҒҜпјҢй“ҫи·ҜеұӮиҝҳдјҡеҠ дёҠе°ҫйғЁдҝЎжҒҜпјҢдҫқж¬Ўе®һзҺ°жҲ‘д»¬жүҖжңүеұӮзҡ„ж•°жҚ®е°ҒиЈ…пјҢжңҖз»ҲдёәеҲ°иҫҫзҪ‘з»ңжҸҗдҫӣжүҖжңүзҡ„еҝ…иҰҒдҝЎжҒҜгҖӮ

ж•°жҚ®е°ҒиЈ…зҡ„иҝҮзЁӢпјҡ

ж•°жҚ®и§Је°ҒиЈ…зҡ„иҝҮзЁӢпјҡ

HTTPж•°жҚ®дј иҫ“иҝҮзЁӢпјҡ

еҸ‘йҖҒз«ҜеҸ‘йҖҒж•°жҚ®ж—¶пјҢж•°жҚ®дјҡд»ҺдёҠеұӮдј иҫ“еҲ°дёӢеұӮпјҢдё”жҜҸз»ҸиҝҮдёҖеұӮйғҪдјҡжү“дёҠиҜҘеұӮзҡ„еӨҙйғЁдҝЎжҒҜгҖӮиҖҢжҺҘж”¶з«ҜжҺҘж”¶ж•°жҚ®ж—¶пјҢж•°жҚ®дјҡд»ҺдёӢеұӮдј иҫ“еҲ°дёҠеұӮпјҢдј иҫ“еүҚдјҡжҠҠдёӢеұӮзҡ„еӨҙйғЁдҝЎжҒҜеҲ йҷӨгҖӮ

# TCP дёүж¬ЎжҸЎжүӢ

дҪҝз”Ё TCP иҝӣиЎҢйҖҡдҝЎзҡ„еҸҢж–№еҝ…йЎ»е…Ҳе»әз«ӢиҝһжҺҘпјҢз„¶еҗҺжүҚиғҪејҖе§Ӣдј ж•°жҚ®гҖӮдёәдәҶзЎ®дҝқиҝһжҺҘеҸҢж–№еҸҜйқ жҖ§пјҢеңЁеҸҢж–№е»әз«ӢиҝһжҺҘж—¶пјҢTCP еҚҸи®®йҮҮз”ЁдәҶдёүж¬ЎжҸЎжүӢзӯ–з•ҘгҖӮ

з¬¬дёҖж¬ЎжҸЎжүӢпјҡ

е®ўжҲ·з«ҜеҸ‘йҖҒеёҰжңү SYN ж Үеҝ—зҡ„иҝһжҺҘиҜ·жұӮжҠҘж–Үж®өпјҢз„¶еҗҺиҝӣе…Ҙ SYN_SENT зҠ¶жҖҒпјҢзӯүеҫ…жңҚеҠЎз«ҜзЎ®и®ӨгҖӮ

з¬¬дәҢж¬ЎжҸЎжүӢпјҡ

жңҚеҠЎз«ҜжҺҘж”¶еҲ°е®ўжҲ·з«Ҝзҡ„ SYN жҠҘж–Үж®өеҗҺпјҢйңҖиҰҒеҸ‘йҖҒ ACK дҝЎжҒҜеҜ№иҝҷдёӘ SYN жҠҘж–Үж®өиҝӣиЎҢзЎ®и®ӨгҖӮеҗҢж—¶иҝҳиҰҒеҸ‘йҖҒиҮӘе·ұзҡ„ SYN иҜ·жұӮдҝЎжҒҜгҖӮжңҚеҠЎз«Ҝдјҡе°ҶдёҠиҝ°дҝЎжҒҜж”ҫеҲ°дёҖдёӘжҠҘж–Үж®өпјҲSYN + ACKжҠҘж–Үж®өпјүдёӯпјҢдёҖе№¶еҸ‘йҖҒз»ҷе®ўжҲ·з«ҜпјҢжӯӨж—¶жңҚеҠЎз«Ҝе°Ҷдјҡиҝӣе…Ҙ SYN_RECV зҠ¶жҖҒгҖӮ

з¬¬дёүж¬ЎжҸЎжүӢпјҡ

е®ўжҲ·з«ҜжҺҘж”¶еҲ°жңҚеҠЎз«Ҝзҡ„ SYN + ACK жҠҘж–Үж®өеҗҺпјҢдјҡеҗ‘жңҚеҠЎз«ҜеҸ‘йҖҒ ACK зЎ®и®ӨжҠҘж–Үж®өпјҢиҝҷдёӘжҠҘж–Үж®өеҸ‘йҖҒе®ҢжҜ•еҗҺпјҢе®ўжҲ·з«Ҝе’ҢжңҚеҠЎз«ҜйғҪдјҡиҝӣе…Ҙ ESTABLISHED зҠ¶жҖҒпјҢе®ҢжҲҗ TCP дёүж¬ЎжҸЎжүӢгҖӮ

# вҖңдҪ жҳҜеҰӮдҪ•и®ҝй—®ж…•иҜҫзҡ„вҖқ--DNSеҹҹеҗҚи§Јжһҗ

жҲ‘д»¬и®ҝй—®дёҖдёӘзҪ‘з«ҷж—¶дёҖиҲ¬жҳҜйҖҡиҝҮеҹҹеҗҚи®ҝй—®зҡ„гҖӮеӣ дёәзӣёеҜ№дәҺ IP ең°еқҖпјҢеҹҹеҗҚжӣҙе®№жҳ“и®©дәәи®°дҪҸгҖӮдҪҶжҳҜ TCP/IP еҚҸи®®дҪҝз”Ёзҡ„жҳҜ IP ең°еқҖиҝӣиЎҢи®ҝй—®зҡ„пјҢжүҖд»Ҙеҝ…йЎ»жңүдёӘжңәеҲ¶жҲ–жңҚеҠЎжҠҠеҹҹеҗҚиҪ¬жҚўжҲҗ IP ең°еқҖгҖӮDNS жңҚеҠЎе°ұжҳҜз”ЁжқҘи§ЈеҶіиҝҷдёӘй—®йўҳзҡ„пјҢе®ғжҸҗдҫӣеҹҹеҗҚеҲ° IP ең°еқҖд№Ӣй—ҙзҡ„и§ЈжһҗжңҚеҠЎгҖӮ

е…Ҳд»Һжң¬ең° host ж–Үд»¶еҜ»жҹҘ IP ең°еқҖпјҢеҰӮжһңжүҫеҲ°е°ұзӣҙжҺҘдҪҝз”Ё host ж–Үд»¶дёӯзҡ„ IP ең°еқҖгҖӮеҰӮжһңжң¬ең°жІЎжңүеҜ№еә”еҹҹеҗҚзҡ„ IP ең°еқҖжҳ е°„пјҢиҝҷдёӘж—¶еҖҷдјҡеҺ»жң¬ең°зҡ„ DNS жңҚеҠЎеҷЁпјҢжҹҘжүҫеҹҹеҗҚеҜ№еә”зҡ„ IP ең°еқҖпјҢеҰӮжһңжң¬ең°зҡ„ DNS жІЎжңүжүҫеҲ°пјҢдјҡдёҖеұӮеұӮеҗ‘дёҠдёҖеұӮ DNS жңҚеҠЎеҷЁеҸ‘йҖҒиҜ·жұӮпјҢзӣҙеҲ° DNS ж №жңҚеҠЎеҷЁпјҢе°ҶжүҫеҲ°зҡ„ IP ең°еқҖиҝ”еӣһз»ҷе®ўжҲ·з«ҜгҖӮ

# еӣһжәҜHTTPдәӢеҠЎеӨ„зҗҶиҝҮзЁӢ

еҪ“е®ўжҲ·з«Ҝи®ҝй—® Web з«ҷзӮ№ж—¶пјҢйҰ–е…ҲдјҡйҖҡиҝҮ DNS жңҚеҠЎжҹҘиҜўеҲ°еҹҹеҗҚзҡ„ IP ең°еқҖгҖӮз„¶еҗҺжөҸи§ҲеҷЁз”ҹжҲҗ HTTP иҜ·жұӮпјҢе№¶йҖҡиҝҮ TCP/IP еҚҸи®®еҸ‘йҖҒз»ҷ Web жңҚеҠЎеҷЁгҖӮWeb жңҚеҠЎеҷЁжҺҘж”¶еҲ°иҜ·жұӮеҗҺдјҡж №жҚ®иҜ·жұӮз”ҹжҲҗе“Қеә”еҶ…е®№пјҢе№¶йҖҡиҝҮ TCP/IP еҚҸи®®иҝ”еӣһз»ҷе®ўжҲ·з«ҜгҖӮ

иҜҰз»ҶзӮ№зҡ„еӣҫи§Јпјҡ

# гҖҗи®Ёи®әйўҳгҖ‘жөҸи§ҲеҷЁиҫ“е…Ҙең°еқҖеҲ°йЎөйқўеұ•зӨәпјҢдёӯй—ҙжҖ»е…ұз»ҸеҺҶдәҶд»Җд№Ҳпјҹ

вҖӢ1.жөҸи§ҲеҷЁиҫ“е…ҘurlгҖӮе…Ҳи§Јжһҗurlең°еқҖжҳҜеҗҰеҗҲжі•гҖӮ 2. жөҸи§ҲеҷЁжЈҖжҹҘжҳҜеҗҰжңүзј“еӯҳпјҲжөҸи§ҲеҷЁзј“еӯҳ-зі»з»ҹзј“еӯҳ-и·Ҝз”ұеҷЁзј“еӯҳпјүгҖӮеҰӮжһңжңүпјҢзӣҙжҺҘжҳҫзӨәгҖӮеҰӮжһңжІЎжңүпјҢи·іеҲ°з¬¬дёүжӯҘгҖӮ 3. еңЁеҸ‘йҖҒhttpиҜ·жұӮеүҚпјҢйңҖиҰҒеҹҹеҗҚи§ЈжһҗпјҲDNSи§ЈжһҗпјүпјҢи§ЈжһҗиҺ·еҸ–еҜ№еә”иҝҮзҡ„ipең°еқҖгҖӮ 4. жөҸи§ҲеҷЁеҗ‘жңҚеҠЎеҷЁеҸ‘иө·tcpй“ҫжҺҘпјҢдёҺжөҸи§ҲеҷЁз®ҖеҺҶtcpдёүж¬ЎжҸЎжүӢгҖӮ 5. жҸЎжүӢжҲҗеҠҹеҗҺпјҢжөҸи§ҲеҷЁеҗ‘жңҚеҠЎеҷЁеҸ‘йҖҒhttpиҜ·жұӮпјҢиҜ·жұӮж•°жҚ®еҢ…гҖӮ 6. жңҚеҠЎеҷЁж”¶еҲ°еӨ„зҗҶзҡ„иҜ·жұӮпјҢе°Ҷж•°жҚ®иҝ”еӣһиҮіжөҸи§ҲеҷЁгҖӮ 7. жөҸи§ҲеҷЁж”¶еҲ°httpе“Қеә”гҖӮ 8. жөҸи§ҲеҷЁи§Јжһҗе“Қеә”гҖӮеҰӮжһңе“Қеә”еҸҜд»Ҙзј“еӯҳпјҢеҲҷеӯҳе…Ҙзј“еӯҳгҖӮ 9. жөҸи§ҲеҷЁеҸ‘йҖҒиҜ·жұӮиҺ·еҸ–еөҢе…ҘеңЁHTMLдёӯзҡ„иө„жәҗпјҲhtmlпјҢcssпјҢJavaScriptпјҢеӣҫзүҮпјҢйҹід№җзӯүпјүпјҢеҜ№дәҺжңӘзҹҘзұ»еһӢпјҢдјҡеј№еҮәеҜ№иҜқжЎҶгҖӮ 10. жөҸи§ҲеҷЁеҸ‘йҖҒејӮжӯҘиҜ·жұӮгҖӮ 11. йЎөйқўе…ЁйғЁжёІжҹ“з»“жқҹгҖӮ
