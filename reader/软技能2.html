# 书摘

  • 工作是属于公司的，而职业生涯却是属于你自己的。

# 读后感

# 第三章 如何给自己设定好职业目标

# 书摘

  • 你应该定期追踪并更新自己的设定的目标，以防出错。
  • 在生活中，即使选择的是次优的选项或者次优的路径，那也几乎总是比随波逐流的路径更好。
  • 如果你准备达成一个大目标，却没有分解为小目标，那么当你偏离航向时也不会有时间去校正。
  • 不要成为大海上漫无目的随波逐流的船。启航前总要规划好航海路线。

# 读后感

我们每个人都应该给自己设定目标，随波逐流就是浪费生命。

# 第四章 拓展自己的人际交往能力

# 书摘

  • 每个人都希望感到自己很重要。
  • 如果你不能保全他人的自尊，那你永远也不可能赢得他的心。
  • 如果你因为“该怎么做某事”与他人发生分歧，那么在很多情况下，你最好先确定分歧点是否值得你去拼命维护。

# 读后感

在和别人聊天的过程中要适度尊重他人，不要一直否认他人，如果发生争论，要仔细想想有没有违背自己的底线，如果没有就尽量不要争论，赢了也没什么意义，还容易得罪人。

# 第五章 创建一份屡试屡验的简历

# 读后感

这一章主要讲的是写简历。作者说可以雇人写简历，这个我个人不是很赞同，作为一个找工作的人，如果你连简历都不是自己写的，怎么能确保你能应对工作中的困难。但是请人润色润色我感觉还是可以的。

# 第六章 破解面试之道

# 书摘

  • 大多数面试官决定雇佣某个人其实是基于各种各样的非技术因素。
  • 通过面试的最快捷的方式是让面试官对你怀有好感。
  • 我是那种能够主动找到自己需要做什么，也能自觉主动弄清楚怎么做的人。

# 第七章 软件开发人员的三条职业路径

# 读后感

在入想从事的职业前，最好先找从事这个职业的人了解下这条路到底怎么样。

# 第八章 为什么你需要走专业化道路

# 书摘

  • 专业化的规则是：专业化程度越深，潜在的机会就越少，但获得这些机会的可能性越大。
  • 只要你专业能力雄厚，市场没有过度饱和，与那些自称“软件开发人员”的人相比，你就能更轻松的找到工作或赢得客户。

# 读后感

在一个专业方向上拥有专长，而不是语言。

# 第九章 公司与公司是不一样的

# 书摘

大公司工作的好处：

  • 完备的流程和规范
  • 培训机会多
  • 大型有影响力的项目

大公司的坏处：

  • 充斥着官僚主义作风
  • 可能只负责代码库的一小部分
  • 很难获得关注

# 第十章 攀登晋升阶梯

# 书摘

  • 在任何公司能让你脱颖而出的最重要的法宝就是承担更多的责任。

# 读后感

在公司我们应该怎么晋升呢？首先要承担更多责任，在合适的场合增加自己的曝光度，比如会议上或者技术分享等。努力自学知识，让自己变的更有价值。提升自己解决问题的能力。还要注意，如果你不懂办公室政治，就要尽量远离，不要瞎掺和，需要知道哪些人需要避开，哪些人永远不要有交集。

# 第十一章 成为专业人士

# 书摘

  • 如果你想成为专业人士，就要致力于持续不断地完善自己的技能，学习更多与专业先关的匠艺，确保自己制定并执行了学习计划，拓展自己的技能、学习新东西，这将有助于你做得更好。不要沾沾自喜，永远对努力成为更好的自己心存渴望。
  • 你做的每一件事情就是你所做的一切。-《百万富翁的秘密》
  • 当你需要一份工作的时候，就很容易落入圈套。一旦陷入困境，你就限制了自己的选择，任由他人给你施加压力。
  • 为了达到自己在工作中所需要的专业化水平，你得明确自己要养成哪种习惯。习惯至关重要，因为它让你养成持之以恒的品质，而这一品质让你成为值的信赖的人。
  • 每天我们都有“每日站立会议”，说明自己已经做了什么、计划做什么、有哪些障碍。

# 第十二章 与老板和同事的相处之道

# 书摘

  • 生活中有些东西是你能掌控的，有些是你无法掌控的，你能掌控的东西太少了，而你的态度和看法是为数不多你一直可以掌握的东西。
  • 与老板和同事相处的好坏至关重要，这直接决定了你的工作体验是乐在其中还是度日如年。

  • 刚开始的时候要从小处着手，作为副业开始。

# 第二十七章 学习怎样学习

# 书摘

  • 学校并不会教你自学这项技能。
  • 教育的首要目标，并不在于知，而在于行。

# 读后感

学习的最有效方式就是带着问题去学习，学完抓紧用到实践中，最好还能把学到的东西教给别人。

# 第二十八章 我的十步学习法

# 书摘

  • 我很自然会先明确需要掌握哪些内容，再去寻找我能获得这些信息的最佳资源，同时也会忽略那些并非达成目标所必需的其他信息。
  • 我发现，如果我了解三项主题——如何开始，我能做什么，以及相关基础知识，那么我就能随着学习深入学会所需的其他知识。
  • 在掌握一门知识前我们需要明确三个目标：
    • 如何开始 - 需要学哪些才能尽快应用到实践中
    • 学科范围 - 熟悉轮廓，但不必了解所有细节
    • 基础知识 - 掌握20%可以应对80%的使用
  • 通过“学习—实践—掌握—教授”的过程，获得对该学科的深刻理解。
  • 十步学习法：
    • 了解全局
    • 确定范围
      • 在这个过程中最容易犯的错是扩展，看到一个知识点又想去学它，然后偏离了既定目标。
    • 定义目标
      • 目标一定要清晰，具体可执行，不能有二义性。
    • 寻找资源

    • 创建学习计划
    • 筛选资源
    • 开始学习，浅尝辄止
    • 动手操作，边玩边学
    • 全面掌握，学以致用
    • 乐为人师，融会贯通
  • 十步学习法只是通过“聚焦学习范围，让你关注重点内容”来学习的一种快速学习方法，它迫使你通过“自我探索”和“主动教学”的方式将真正重要的内容印刻在自己的大脑中。

# 读后感
