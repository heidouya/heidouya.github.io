# 推荐序二 兴趣变现是最好的起点

# 书摘

  • 创作与目标群体相关的内容是成功创业的好办法。
  • “最小可行受众”是指，当你的受众达到一定数量后，就可以通过他们自发的社交分享和口碑带来自然增长。与此同时，你还会开始收到有关受众想要购买产品和服务的反馈。
  • 最安全的创业方式-内容创业
  • 自媒体的蓬勃发展，赋予每个人发声的权利。
  • 为个体赋能是这个时代成功企业的新使命。

# 第一章 内容营销之父的非典型创业

# 书摘

  • 他发现通过创作和发布有价值的、有相关性的、持续性的内容，能够有效吸引目标人群，并强化他们的信任，最后将这些信任行为转化为消费行为。

  • 最低风险的创业不再是从产品开始，而是要从你最热爱和最感兴趣的领域出发，创建一个吸引粉丝的内容分享机制。借助内容的力量，只要你吸引到一群信任你的粉丝，就完全可以养活自己。

  • 要让内容本身具有传播力，而不需要去花钱推广。

  • 用良好的内容体验建立忠实社群，而非一味推销既有产品

# 第二章 找到你的甜蜜点

# 书摘

  • 玩也是一种生产力。
  • 甜蜜点就是知识领域与兴趣点的交集。

怎么确立“甜蜜点”：

  1. 找到自己擅长的知识或技能
  2. 找到你的兴趣所在
  3. 以用户价值激活甜蜜点

确认目标用户的三个问题：

  1. 他是做什么的
  2. 他有什么问题需要解决
  3. 他为什么会关注你，关注你的产品和服务

# 什么是内容翘点

翘点的两层含义：

  • 与众不同的印象

假如你的内容在一夜间消失，就像从来没有存在过。会有人想念它么？
设定奋斗目标无法获得粉丝关注，只有找到”内容翘点“，然后在你的领域做到最好才行。

# 第六章 如何创造爆款内容

# 六条行之有效的营销守则

  • 满足需求：可以帮用户解决问题
  • 持之以恒：定期更新，和用户保持联系
  • 人格魅力：独特的风格可以增强用户记忆
  • 抒发己见：输出的内容不需要绝对客观，这样增加用户参与度
  • 避免推销腔：很少有人喜欢别人给自己推销东西
  • 争做业界第一：提高自己的目标，可以帮助自己更好的提升产品

# 怎样创作有价值的内容

  • 被忽视的内容盘点：比如你的朋友圈里的一些内容是可以加工重新发的
  • 50个问题，深度洞察用户需求：通过问问题或者寻找用户的痛点来优化内容，为用户提供价值
  • 百度新闻订阅是灵感仓库：要多方拓展内容渠道
  • 分析粉丝每日阅读数据：分析用户各种数据来优化内容
  • 案例访谈的技巧：舒服的沟通的方式可以获得更多用户信息
  • 定期组织员工讨论：说白了就是集思广益
  • 看一本毫不相关的书：从不同的方面寻找灵感
  • 学会自由写作：通过这种手段可以激发灵感

# 第十一章 建立个人品牌的三三模型

# 前一个三，关于个人规划

建立个人品牌三条策略：自媒体、书、演讲。

关于写书的建议：

  1. 做一个深度的内容盘点：平时要多注意积累，写之前可以翻看下自己之前写的东西
  2. 你的自媒体是个宝藏
  3. 与行业伙伴联合创作
  4. 明确使命——希望读者从书中得到什么？
  5. 借用意见领袖的传播力
  6. 可以考虑代写
  7. 你需要编辑的帮助
  8. 怎样规划一本书的内容？
    1. 封面内容、概念介绍、概念总结、章节大纲、关于作者、作者的基本观点、图书营销、读者是谁、竞争、补充介绍

关于演讲的建议：

  1. 筹备演讲阶段，个人网站是知识储备的背书
  2. 暂时没机会？那就从演讲视频开始
  3. 设置一个记忆的钩子
    1. 确定一种颜色：增强记忆点
    2. 参考电梯理论：围绕自己擅长的东西把事情讲清楚

演讲规划：

  1. 创建活动清单：知道自己要去哪些地方演讲
  2. 通过适当的渠道接触主办方
  3. 除非你的观点鲜明，否则不要在公开场合张口就说。
  4. 重点突出
  5. 资讯要新：采用的例子要新鲜，不要用太老的
  6. 辅助材料要简洁，不要花里胡哨的
  7. 讲故事：要有趣
  8. 不要辩护：我们的目的是赢得朋友不是炫耀
  9. 帮助听众解决问题
  10. 不用刻意追求有趣的开头结尾
  11. 把要点总结成简洁、令人信服的材料
